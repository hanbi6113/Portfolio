# Todo List 프로젝트

## 1. 프로젝트 소개 
이번 프로젝트는 React를 활용하여 간단한 Todo List를 구현한 과제입니다. 사용자가 할 일을 추가하고 상태를 변경하며, 검색과 필터링 기능을 통해 효율적으로 작업을 관리할 수 있도록 하였으며, 로딩 시 진행 상황을 게이지 형태로 보여주는 UI와 에러 처리 UI도 포함했습니다. 
(로딩 시간을 1초 동안이라고 쓰여있었지만 실제 웹 느낌 나게끔 게이지, 3초로 구현했습니다.)

프로젝트 실행 방법은 GitHub에서 레포지토리를 클론한 후 `npm install`로 필요한 패키지를 설치하고, `npm start`로 개발 서버를 실행하면 브라우저에서 `http://localhost:5173/`으로 확인할 수 있습니다.

과제 진행 중에는 상태 관리, 필터링, 검색 기능과 로딩 UI, 에러 UI 구현에 특히 신경을 썼으며, 시간이 더 있었다면 로컬 스토리지나 서버 연동을 통한 데이터 영구 저장, 드래그 앤 드롭을 이용한 Todo 순서 변경 기능, UI/UX 개선과 모바일 대응 등을 추가로 구현하고 싶었습니다.
추가로 관리자페이지로 사용이 가능한 웹으로 만들고 싶습니다.  

과제는 약 2시간 내외로 설계되었고, 실제로는 약 3시간 정도 소요되었습니다.

특징:
- 상태별 작업 관리: 할 일, 진행 중, 완료  
- 상태별 커스텀 분류순 정렬: "할 일 → 진행 중 → 완료" 순서  
- 분류 기능도 추가: 기본순이 기본으로 뜨게
- 검색 기능으로 작업 제목으로 필터링 가능  
- 상태별 필터 버튼으로 특정 상태만 확인 가능  
- 직관적인 UI와 상태별 색상 적용  
- 간단한 로딩 게이지 적용으로 초기 로딩 시 시각적 딜레이 제공  

사용한 기술은 React 18, Tailwind CSS, 최신 JavaScript(ES6+), Node.js 및 npm, Git/GitHub를 활용하였으며, UI와 상태 관리에 집중했습니다.
