# Todo List 프로젝트

## 1. 프로젝트 소개
이 프로젝트는 간단한 Todo List입니다.  
사용자가 할 일을 추가하고, 진행 상태를 변경하며, 검색과 필터링 기능으로 작업을 관리할 수 있도록 설계했습니다.  

특징:
- 상태별 작업 관리: `할 일`, `진행 중`, `완료`  
- 상태별 커스텀 분류순 정렬: "할 일 → 진행 중 → 완료" 순서  
- 분류 기능도 추가: 기본순이 기본으로 뜨게
- 검색 기능으로 작업 제목으로 필터링 가능  
- 상태별 필터 버튼으로 특정 상태만 확인 가능  
- 직관적인 UI와 상태별 색상 적용  
- 간단한 로딩 게이지 적용으로 초기 로딩 시 시각적 딜레이 제공  

프로젝트는 React와 TailwindCSS를 사용하여 구현했으며, UI와 상태 관리에 집중했습니다.
